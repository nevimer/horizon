round_event_control
/datum/round_event_control

Vars

calculated_weightLast calculated weight that the storyteller assigned this event
can_run_post_roundstartWhether a roundstart event can happen post roundstart. Very important for events which override job assignments.
costHow much event points will this event spend. It's a multiplier to the track threshold and effectively affects how long until the next event of the same track type. Affected by random bell curve for variance.
min_eng_crewMinimum engineering crew required for the event to spawn
min_head_crewMinimum head role crew required for the event to spawn
min_med_crewMinimum medical crew required for the event to spawn
min_sec_crewMinimum security crew required for the event to spawn
reoccurence_penalty_multiplierMultiplier to the penalty applied to weight for re-occurance. The smaller the lesser the penalty
roundstartWhether this is a roundstart event or not. Not exactly sure how I should handle this.
shared_occurence_typeWhether this event uses a shared occurence type, sharing occurence counts with the ones that also do have this set. Important if you want different event types to share weight penalties for recurrence.
shared_occurencesList of the shared occurence types.
tagsTags of the event
trackTo which event track does this event belong to

Procs
