attribute_holder
/datum/attribute_holder

Holder datum for attributes, skill and modifiers for them. Those get attached to mob/living

Vars

attributes_finalAssoc list of attribute types to final values (with buffs)
attributes_rawAssoc list of attribute types to raw values
buffsAssoc list, from buff ID to instances of /datum/attribute_buff
ownerMob owner of this attribute holder.
skills_finalAssoc list of skill types to final values (with buffs)
skills_rawAssoc list of skill types to raw values

Procs

add_attribute_sheetAdds values from an attribute sheet to this holder
add_attributesAdd or subtract attributes to the holder from a list
add_buffInstantiates a buff from a type and adds id to the holder.
add_buff_instanceAdds an already instantiated buff to the holder. Useful if you need to customise a buff before adding it.
add_skillsAdds or subtract skills to the holder from a list
attach_buffInternal proc to actually add a buff
get_buff_fromReturns a buff from a given ID, possibly useful to modify existing buffs, just remember to call update_attributes() afterwards.
has_buff_fromWhether the holder has a buff from a given ID
remove_attribute_sheetRemoves values from an attribute sheet from this holder
remove_buffRemoves a buff of a given ID
show_admin_edit_panelBelow is admin panel code for editing the attributes in a holder
update_attributesUpdates final values of the attributes and skills.
